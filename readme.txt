== Description ==

If you wish to start a coin/ico directory this plugin will help you. This plugin publish around 1500 coins/ico information using shortcode

We here using updates keep updating new coin/ico so that your directory remain updated.
 
you only need to insert the shortcode [AistoreCoinCurrDirectory] all other work will be managed by this plugin.


 
= Usage =

1. Download and extract   to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. "add page"->"in content "->"use code [AistoreCoinCurrDirectory]"
 



== Installation ==

1. Download and extract   to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Insert the shortcode where you wish to show [AistoreCoinCurrDirectory]
